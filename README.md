# LIVRA
Sistema de gerenciamento de biblioteca desenvolvido em Java, com funcionalidades de cadastro e controle de usuários, livros e empréstimos.

O projeto aplica conceitos de Programação Orientada a Objetos (POO), utilizando herança, polimorfismo, encapsulamento e manipulação de objetos para representar diferentes tipos de usuários (alunos, professores e usuários comuns) e suas respectivas regras de utilização.

Funcionalidades implementadas:

Cadastro de usuários com diferentes categorias;
Cadastro e listagem de livros;
Busca de usuários e livros por identificação;
Sistema de aluguel e devolução de livros;
Controle de disponibilidade dos livros;
Aplicação de descontos conforme o tipo de usuário:
Alunos: 10%;
Professores: 20%;
Usuários comuns: sem desconto.

O projeto foi desenvolvido com foco na prática de conceitos fundamentais da linguagem Java e na construção de uma aplicação orientada a objetos com fluxo de interação via console.
